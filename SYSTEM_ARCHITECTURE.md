```

   в”ңв”Җ Extract Authorization header
   в””в”Җ Route to Camel

   в”ңв”Җ Extract request body
   в”ңв”Җ Set DTO class header
   в”ңв”Җ Deserialize JSON
   в”ңв”Җ Validate fields
   в”ңв”Җ Call processor
   в””в”Җ Return to client

   в””в”Җ Call DoctorServiceImp.addDoctorSchedule()

   в”ңв”Җ Get current user from SecurityContext
   в”ңв”Җ Find doctor by user ID
   в”ңв”Җ Validate dayOfWeek
   в”ңв”Җ Create DoctorSchedule entity
   в”ңв”Җ Call repository.save()

   в””в”Җ Return saved entity

   в””в”Җ Send to client
```
